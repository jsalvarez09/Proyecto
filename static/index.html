🎓 Motor de Recomendación Docente

Asignación inteligente de profesores según el perfil colectivo del grupo estudiantil. Algoritmo de puntuación ponderada con pesos dinámicos.

Ingresar al sistema

¿Cómo funciona?

👨‍🎓
1. Crear grupo
El coordinador crea la materia y selecciona los estudiantes que la cursan.
📊
2. Perfil colectivo
El sistema calcula automáticamente las métricas del grupo: edad, rendimiento, modalidad.
⚖️
3. Pesos dinámicos
El algoritmo ajusta la importancia de cada criterio según el tipo de grupo detectado.
🏆
4. Recomendación
Se presenta el profesor más compatible con justificación detallada y top 3 alternativas.
